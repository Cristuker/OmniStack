# OmniStack

## Projeto da semana OmniStack da Rockeseat

Tindev é um tinder para desenvolvedores, com a função de like e dislike que usa os dados do Github para isso.



### Tecnologias ultilizadas
- Web: ReactJS
- Backend: NodeJS
- API do GitHub
